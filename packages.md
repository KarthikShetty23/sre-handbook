# Package Management (apt)

**APT** (Advanced Package Tool) is a tool we use to install, update and remove software from Ubuntu.

## Essential Commands

| Command | Description |
| :--- | :--- |
| `sudo apt update` | **ALWAYS RUN FIRST.** Updates the list of available software. |
| `sudo apt install <name>`| Installs a new Package( e.g., `git`, `Python3`). |
| `sudo apt remove <name>` | Removes an Installed Package. |
| `sudo apt list --installed` | Lists all software currently installed in the System. |
